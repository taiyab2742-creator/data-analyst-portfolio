# Data Analyst Portfolio

A personal portfolio website for a junior data analyst, built with HTML, CSS, and JavaScript.

## Overview

This repository contains a responsive portfolio site that showcases analytics skills, data projects, and contact information. It includes a separate build inquiry page, an interactive project gallery, and GitHub Pages publishing support.

## Pages

- `index.html` — Home page with hero section, featured projects, services, and contact details
- `about.html` — About page with profile and career overview
- `projects.html` — Full data analysis project portfolio with filtering
- `resume.html` — Resume page with skills, tools, and timeline
- `contact.html` — Contact page with Netlify form and social links
- `build.html` — Build inquiry page for analytics projects

## Features

- Responsive layout using CSS Grid and Flexbox
- Smooth scrolling and section reveal animations
- Project modal interaction for details and external links
- Netlify form support for contact and project briefs
- Social media links for WhatsApp, Facebook, and Instagram

## How to use

1. Open `index.html` in a browser to view the site locally.
2. Use `projects.html` to explore the full project portfolio.
3. Submit inquiries through `contact.html` or `build.html`.
